<ul>
<li><p>Angular.json file will contain all the configurations of the app.</p>
</li>
<li><p>The builder looks at this file to find the entry point (main.ts) of the application</p>
</li>
<li><p>The main.ts file creates a browser environment for the application to run</p>
</li>
<li><p>calls a function called bootstrapModule, which bootstraps the application via AppModule (declared in the app.module.ts file. )</p>
</li>
<li><p>AppModule contains declarations of all the components, this will bootstrap AppComponenet (defined in app.component.ts file)</p>
</li>
<li><p>AppComponent interacts with the webpage and serves data to it.</p>
</li>
<li><p>Each component is declared with three properties:</p>
</li>
<li><p>Selector - used for accessing the component</p>
</li>
<li><p>Template/TemplateURL - contains HTML of the component</p>
</li>
<li><p>StylesURL - contains component-specific stylesheets</p>
</li>
<li><p>After this, Angular calls the index.html file. This file consequently calls the root component that is app-root. The root component is defined in app.component.ts.</p>
</li>
<li><p>Read more about Angular components, modules, and services <a href="https://mehtalogy.blogspot.com/2020/11/angular-components-modules-and-services.html">here</a>.</p>
</li>
</ul>
